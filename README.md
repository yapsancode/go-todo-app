# Go Todo App

A simple Todo List REST API built with **Go (Gin, GORM, PostgreSQL)**.  
This project is part of my backend learning journey.

## Features
- User registration & login (JWT auth)
- CRUD for todos
- PostgreSQL database with GORM
- Dockerized for easy setup

## Tech Stack
- Go 1.22+
- Gin (HTTP framework)
- GORM (ORM for DB)
- PostgreSQL
- Docker

## Running the App
```bash
go run cmd/main.go
